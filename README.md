# 網路假期查詢瀏覽器插件

本項目為[網路假期 - 解答「新增」和「查詢」按鈕添加](https://github.com/NHDAS/chrome-extension_answer-query-button-add)的分支

-   使用 Typescript 重寫
-   修復類型錯誤
-   新增對 Firefox 的支援

> Google Chrome+Firefox 瀏覽器插件，在網路假期題目頁面「繳卷」和「結束離開」按鈕旁新增一個「解答查詢」和「新增解答至資料庫」的按鈕，讓您不必再複製題目，快速查詢和新增解答！
