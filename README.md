# 網路假期 - 解答「新增」和「查詢」按鈕添加

Google Chrome+Firefox 瀏覽器插件，在網路假期題目頁面「繳卷」和「結束離開」按鈕旁新增一個「解答查詢」和「新增解答至資料庫」的按鈕，讓您不必再複製題目，快速查詢和新增解答！

## 安裝
